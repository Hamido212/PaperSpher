## Dolmetschernetz — Professional Interpreters on Demand

Dolmetschernetz connects qualified interpreters with hospitals, law firms, companies, and private individuals across Germany. The platform replaces slow agency calls with instant booking, transparent reviews, and secure payment.

### Core Features

- **Instant Booking** — Booked within minutes, no waiting
- **Payment Protection** — Payment released only after successful delivery
- **Rating System** — Transparent interpreter profiles with certifications
- **Video Interpreting** — Flexible remote sessions directly via the platform
- **Emergency Button** — Urgent interpreter needed? Connected in under 2 minutes
- **Smart Search** — Filter by language, availability, and location

### Who Is It For?

Hospitals, law firms, companies, and private individuals navigating government processes.

### Tech Stack

Next.js (App Router) with TypeScript. Booking and payment logic via Server Actions, PostgreSQL database. Two user types (client & interpreter) with separate registration flows.
